# Conta Bancária – Console App (C# .NET)

Projeto de estudo desenvolvido em **C# .NET** que simula uma conta bancária simples por meio de uma aplicação **Console**.

O objetivo é praticar conceitos de **POO**, aplicando validações e separação de responsabilidades.

---

## Funcionalidades

- Cadastro de cliente (nome e CPF)
- Depósito de valores
- Saque de valores
- Consulta de saldo
- Exibição dos dados do cliente

## Tecnologias

- C#
- .NET
- Aplicação Console
